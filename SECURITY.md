# Security Policy

## Reporting a Vulnerability

Please report security issues privately and do not open a public GitHub issue.

Use:

- Email: s.werner@sebastian-software.de

When possible, include:

- a description of the issue
- affected versions or commits
- reproduction details
- impact assessment

You should receive an initial response within a reasonable time. Please avoid
public disclosure until the issue has been assessed and a fix or mitigation
plan exists.
